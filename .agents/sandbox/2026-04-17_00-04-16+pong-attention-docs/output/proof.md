# Low-Pong Attention 严格证明

## 1. 任务定义

状态：

\[
s_t = (x_t, y_t, v, p_t),
\]

其中：

- \(x_t \in \{0,\dots,W\}\)
- \(y_t \in \{0,\dots,H-1\}\)
- \(v \in \{-1,+1\}\)
- \(p_t \in \{0,\dots,H-1\}\)

观测：

\[
o_t = (x_t, y_t, p_t).
\]

策略看不到 \(v\)。

转移：

\[
x_{t+1}=x_t-1,\qquad y_{t+1}=y_t+v.
\]

假设从当前时刻到撞到挡板前，不发生顶底反弹。

动作：

\[
a_t \in \{-1,0,+1\},
\qquad
p_{t+1} = \operatorname{clip}(p_t+a_t, 0, H-1).
\]

成功条件：

当 \(x_t=0\) 时，若 \(p_t=y_t\)，则挡球成功。

## 2. 单帧策略不够

### 命题 1

不存在只依赖当前观测 \(o_t\) 的策略 \(\mu(o_t)\)，能在所有可达历史上都最优。

### 证明

取同一个当前观测：

\[
o_t=(x_t,y_t,p_t)=(2,2,2).
\]

历史 A 中 \(y_{t-1}=1\)，于是

\[
v = y_t-y_{t-1}=1,
\qquad
y_{\text{hit}}=2+2\cdot 1=4.
\]

最优动作为 \(+1\)。

历史 B 中 \(y_{t-1}=3\)，于是

\[
v = y_t-y_{t-1}=-1,
\qquad
y_{\text{hit}}=2+2\cdot (-1)=0.
\]

最优动作为 \(-1\)。

两段历史有相同当前观测，但最优动作不同。
所以任何 \(\mu(o_t)\) 都不可能全局最优。

命题得证。

## 3. 两帧历史足够

由无反弹和恒定速度可得：

\[
v = y_t-y_{t-1}.
\]

球还需走 \(x_t\) 步到达挡板，因此拦截点为：

\[
y_{\text{hit}} = y_t + x_t v.
\]

代入得：

\[
y_{\text{hit}} = y_t + x_t(y_t-y_{t-1}).
\]

定义策略：

\[
\pi^\star(h_t)
=
\operatorname{sgn}\!\bigl(y_t + x_t(y_t-y_{t-1}) - p_t\bigr).
\]

### 命题 2

若

\[
|y_{\text{hit}} - p_t| \le x_t,
\]

则 \(\pi^\star\) 能保证最终挡球成功，因此是最优策略。

### 证明

记

\[
d_t = y_{\text{hit}} - p_t.
\]

若 \(d_t>0\)，则动作 \(a_t=+1\)，于是 \(d_{t+1}=d_t-1\)。

若 \(d_t<0\)，则动作 \(a_t=-1\)，于是 \(d_{t+1}=d_t+1\)。

若 \(d_t=0\)，则动作 \(a_t=0\)，于是 \(d_{t+1}=0\)。

所以：

\[
|d_{t+1}| = \max(|d_t|-1,0).
\]

同时：

\[
x_{t+1}=x_t-1.
\]

若起点满足 \(|d_t|\le x_t\)，则归纳得到每一步都满足 \(|d_{t+k}| \le x_{t+k}\)。

当球撞到挡板时 \(x_\tau=0\)，于是

\[
|d_\tau| \le 0 \Longrightarrow d_\tau=0.
\]

即

\[
p_\tau = y_\tau.
\]

所以成功挡球。

命题得证。

结论：对所有 \(t\ge 1\)，最优动作只依赖 \((o_t,o_{t-1})\)。

## 4. attention 可以实现该策略

目标不是让 attention “理解物理”，而是让它完成这件事：

1. 取出当前帧 \(o_t\)
2. 取出上一帧 \(o_{t-1}\)
3. 交给 readout 算出 \(\pi^\star\)

### 命题 3

存在 causal two-head attention，使两个 head 分别近似选择：

- 当前帧
- 上一帧

### 证明

设历史长度有上界，使用 one-hot 位置编码。

对 head A，让 decision token 的 query 对准位置 \(t\)；
对 head B，让 decision token 的 query 对准位置 \(t-1\)。

若 logit scale 为 \(\alpha\)，则目标位置上的 softmax 权重为：

\[
\frac{e^\alpha}{e^\alpha + C},
\]

其中 \(C\) 是非目标位置数量上界。

当 \(\alpha \to \infty\) 时，该权重趋于 1。
所以两个 head 可以分别任意精确地逼近 \(o_t\) 与 \(o_{t-1}\)。

命题得证。

### 命题 4

存在有限 readout，可把这两个被检索出的观测精确映射到最优动作。

### 证明

因为：

- \(x_t\)、\(y_t\)、\(p_t\) 都来自有限集合；
- \(v = y_t-y_{t-1}\) 只取 \(-1\) 或 \(+1\)；

所以

\[
(x_t,y_t,p_t,v)
\]

只取有限多个离散值。

而最优动作

\[
\pi^\star = \operatorname{sgn}(y_t + x_t v - p_t)
\]

是这个有限集合上的确定函数。

任何有限离散函数都可以由有限宽度的 MLP / readout 精确实现。

命题得证。

## 5. 主定理

### 定理

在上述 `Low-Pong` 任务中：

1. 单帧策略不可能全局最优；
2. 两帧策略 \(\pi^\star\) 可以全局最优；
3. 存在 causal multi-head attention 加有限 readout，能精确实现 \(\pi^\star\)。

### 证明

第 1 条由命题 1。

第 2 条由命题 2。

第 3 条由命题 3 与命题 4。

定理得证。

## 6. 结论

这个证明只说明一件事：

> 在这个最小挡球问题里，attention 的价值不是“更大更强”，而是  
> 它能以 causal 的方式，从历史中准确取出当前决策真正需要的 token。
