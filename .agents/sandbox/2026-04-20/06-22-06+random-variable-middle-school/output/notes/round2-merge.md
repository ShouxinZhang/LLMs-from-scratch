# Round 2 Merge: Final Teaching Decisions

## Final Position

Use a three-step teaching path:

1. A real-life motivating question:
   bus waiting time after school.
2. The smallest clean random-variable example:
   one coin flip with
   - Heads -> 1
   - Tails -> 0
3. The main conceptual example:
   two coin flips with
   X = number of heads.

This keeps the opening intuitive, gives a minimal rule-based example early, and then uses the strongest table for distinguishing outcomes from values.

## Decisions

1. Do not use probability notation in the main handout.
2. Introduce the symbol X late on page 1 or early on page 2, only after the idea is clear in words.
3. Cut the weather-coding example from the main path.
4. Keep one non-example:
   Heads -> "happy", Tails -> "sad".
5. Keep one basic fact:
   the same experiment can have different random variables.
6. Use the die only as a later reinforcement example, not the main opening example.

## Final Beginner Sequence

1. Why do we need math for uncertain situations?
2. What is an outcome?
3. Why attach numbers to outcomes?
4. What is a random variable?
5. Is it the same as the outcome, or just a random number?
6. Why is it called a variable?

## Page 1 Requirements

- Plain English first
- No measure-theory language
- No probability notation
- One boxed definition
- One tiny coin table
- One sentence on time order:
  before observing, several values are possible; after observing, one value is fixed
- One warning box:
  a random variable is not the same as the outcome
